## ✦ What is Twilight Optimizer?

Twilight Optimizer is a sleek, **Discord-authenticated** Windows optimization suite designed to squeeze every bit of performance out of your PC. Built with a dark aesthetic and a signature pink accent, it gives you direct control over Windows, GPU, network, and system tweaks — all in one place.

> **Access is gated through your Discord account and role.** You must be a member of the Twilight server with an active subscription to use the app.
